### 반복

- while문을 이용한 반복문      ## 잘 안 쓰임

```python
while n < 3:
	print('출력')
    n = n+1
```

- for문을 이용한 반복문

```python
dust = [10,20,30]
for i in dust:
    print(dust)
```

- **python에서의 흔한 반복문 꼴**

  ex) 100회 반복 시

```python
for i in range(100):
    code
```
